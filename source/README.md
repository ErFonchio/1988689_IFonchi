# 1988689_IFonchi

## User stories

1. As a client i want to see the event name with the corresponding value on the dashboard
2. As a client i want to inspect the historical event on the dashboard
3. As a client i want to download the report of the events.
4. As a user i want to see the filtered/processed data of the event
5. As a client i want to be able to refresh manually and automatically the dashboard
6. As a client i want to see the types of events i can access to.
7. As a client i want to insert in the dashboard a filter based on events/timeline/location.
8. As an admin i want to see how many replicas are alive
9. As an admin i want to log in to the dashboard to see the alive replicas.
